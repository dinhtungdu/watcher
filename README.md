# Watcher

A tmux-native switcher for local coding agents.

I live in the terminal, already know tmux, and do not want another dashboard to babysit. Watcher shows which agent panes need attention, previews enough context, then jumps straight back into tmux.

![Watcher terminal preview](docs/assets/watcher-terminal-preview.png)

```sh
watcher
```

```sh
watcher daemon
watcher integrations install pi
watcher integrations status
```

Pi panes need `/reload` or restart after installing the integration.
